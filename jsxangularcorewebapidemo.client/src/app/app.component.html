jSX Angular Core Web API Demo Project

Phase 1: Weather Forecast

This Angular component demonstrates fetching weather data from the server. However, this initial page is the absolute, bare minimum, vanilla version of the application required in order for it run, typically found in out-of-the-box Visual Studio Angular Client-Side & Asp.NET Core Server-Side Templates & CLIs. This is absolutely by my design.


Although the development Phases may change, there are 4 major phases I have planned, in which I plan to put the jSX Blazor, jSX Angular, jSX React & jSX Vue backed Asp.net Core projects through. This is also a deliberate attempt to show the evolution of typical Minimum Viable Products to Pruduction Ready Products, and when Minimum Viable Products must begin to make the more careful approach to development in very different, thoughtful, careful and secure ways.
